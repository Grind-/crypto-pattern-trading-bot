Das Vier-Agenten-System
Jeder Agent ist auf eine Aufgabe spezialisiert — gemeinsam treffen sie bessere Entscheidungen
Agent 1
📊
Signal Agent
Analysiert Kerzen, Indikatoren (RSI, MACD, Bollinger) und die Wissensbasis — gibt BUY / SELL / HOLD mit Konfidenzscore aus.
Agent 2
🌐
Regime Agent
Erkennt den aktuellen Marktmodus (BULL_TREND, BEAR_TREND, RANGING, HIGH_VOLATILITY) und gewichtet das Signal entsprechend.
Agent 3
📰
News Agent
Liest stündlich aktuelle Krypto-Nachrichten und berechnet einen Sentiment-Score. Bei negativen Ereignissen wird ein Kauf-Veto gesetzt.
Agent 4
🛡️
Risk Agent
Berechnet ATR-basierte Stop-Loss / Take-Profit Levels und die optimale Positionsgröße. Blockiert Käufe wenn weniger als 2/4 Signale grün sind.
